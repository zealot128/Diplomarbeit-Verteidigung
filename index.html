  • iterative Entwicklungstechnik
  • bekannt durch eXtreme Programming (XP)

Red

class TestAdder &lt; Test::Unit::TestCase
  def test_addition
    assert_equal 3, Adder.add(1,2)
  end
end

class Adder
  def self.add(a,b)
    return 3
  end

Refactor

class Adder
  def self.add(a,b)
    a + b
  end
end

def self.add(*args)
  args.sum
end

Agile in a Flash

Vorteile

  • Code und Test entstehen im Einklang
  • → übersichtlichere Methoden
  • von Stunde 0 an hohe Testabdeckung
  • ermöglicht sichere Refaktorisierungen
